FAQ

Frequently asked questions

Answers to the things people ask most.

What does a "credit" mean?
One credit = one organize run. A run scans your inbox, applies your rules, and labels/archives matching emails. You start with 20 free credits — no card needed.
Does it read all my emails?
Only the metadata it needs — sender, subject, and headers. Email body content is never read unless you explicitly ask for a thread summary, and even then it goes directly to Gemini using your own API key.
Will it work on emails I already have?
Yes. Unlike most competitors, Gmail Organizer applies rules retroactively — scanning and labeling your entire inbox history, not just new emails.
What if I regret an organize run?
There's an Undo button that reverses the last run. We also show a full preview before applying any changes, so you always know what will happen before you confirm.
How does the unsubscribe feature work?
It reads the List-Unsubscribe header in newsletters. For mailto unsubscribes, it automatically sends the opt-out from your Gmail. After 7 days it checks whether the sender stopped emailing you.
Do I need a Gemini API key?
Only for AI features like generating rules from a description or summarising threads. All core organizing, unsubscribing, follow-up detection, and bulk actions work without any API key.
Is this affiliated with Google?
No. Gmail Organizer is an independent Chrome extension that uses Gmail's official API with your explicit OAuth permission. It is not affiliated with or endorsed by Google LLC.
