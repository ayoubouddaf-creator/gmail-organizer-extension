FAQ

Common questions

Does Gmail Organizer read all my emails?

No. The extension only reads the metadata needed to match your rules (sender, subject, labels). Email bodies are only accessed when you explicitly trigger AI summarization — and only the selected thread is sent to Gemini.

What are credits used for?

Credits are consumed by AI-powered features like rule generation and thread summarization. Manual rules, unsubscribe, bulk actions, and the priority score are all free and unlimited.

Will unsubscribe actually work?

Yes. The extension sends the unsubscribe request directly via Gmail API (no clipboard, no pop-up). Seven days later it checks whether the sender sent you anything new — and marks the result as confirmed or failed.

Can I undo a run?

Yes. Every run stores a snapshot. Hit Undo in the popup to reverse the last labeling batch instantly.
